Features

Initiative Tracker

  • Keeps track of player's initiative so you do not need to worry about tracking who's turn is next and sorting the turn order!
  • Future features will include a spell slot tracker and status effect markers!

Battle Tracker (Future Feature)

  • Will allow you to create Encounters before hand and add monsters/enemies to them so you can have multiple different scenarios setup before you start each session!
  • After creating an Encounter quickly add monsters from the 5E Monster Manual and other reference material or create your own custom monsters!

Inventory Management (Future Feature)

  • Lets you organize and sort all of your items in one place!
  • Persistent storage for multiple characters!
